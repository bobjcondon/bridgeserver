# Table


Each of the main database tables, (T) has a route (src/routes/T) which displays all the items in T.

Rewrite the route src/routes/players to use a shadcn-svelte data table using the example
in https://shadcn-svelte.com/docs/components/data-table.  

- Sortable columns: { firstName, lastName, email}
- Searchable columns: { firstname, lastname, email}
